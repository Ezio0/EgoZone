# 如何将设备添加到信任列表

## 方法一：自动添加（推荐）

1. 在您希望信任的设备上（如家里的电脑）打开浏览器访问 EgoZone
2. 在登录页面输入正确的密码
3. **勾选"信任此设备"复选框**（重要！）
4. 点击登录
5. 系统会自动记录该设备的信息并将其添加到信任列表

## 方法二：手动获取设备指纹并添加

如果您希望提前将设备添加到信任列表，可以通过以下步骤获取设备指纹：

### 步骤 1: 获取User-Agent字符串
在您家里的电脑上打开浏览器，然后：

- **Chrome/Firefox/Safari**: 按 F12 键打开开发者工具
- 在 Console（控制台）标签页中输入以下代码并按回车：
  ```
  navigator.userAgent
  ```

### 步骤 2: 使用指纹工具添加设备
1. 在EgoZone项目目录中运行以下命令：
   ```
   python get_device_fingerprint.py
   ```

2. 按提示输入刚才获取的 User-Agent 字符串
3. 输入设备的IP地址（如果不知道可输入 "127.0.0.1" 或 "unknown"）
4. 输入设备名称（如"家里的电脑"）

### 步骤 3: 或者直接通过浏览器获取
1. 在目标设备上打开浏览器访问 EgoZone
2. 打开开发者工具 (F12)
3. 转到 Console 标签
4. 输入以下代码获取User-Agent：
   ```
   copy(navigator.userAgent)
   ```
   这会将User-Agent复制到剪贴板

## 验证信任设备是否生效

### 查看所有信任设备
使用以下命令查看已信任的设备：
```
python manage_trusted_devices.py list
```

或通过API访问：
```
GET /api/auth/trusted-devices
```

### 删除信任设备
如果某个设备不再可信，可以删除它：
```
python manage_trusted_devices.py remove --fingerprint <设备指纹>
```

## 重要提示

1. **设备指纹是唯一的**：每个设备的 User-Agent 和 IP 组合会产生唯一的指纹
2. **网络变化影响**：如果您的家庭网络IP发生变化，可能需要重新信任设备
3. **浏览器变化**：不同的浏览器会有不同的 User-Agent，需要分别信任
4. **安全性**：只对您真正控制的设备选择"信任此设备"

## 故障排除

如果设备无法被正确识别为信任设备：

1. 确认User-Agent字符串是否一致
2. 检查是否有代理或VPN改变了网络标识
3. 查看服务器日志中的设备指纹是否与信任列表中的匹配
4. 确保数据文件 `data/trusted_devices.json` 存在且格式正确
